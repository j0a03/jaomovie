# tutorial para usar o projeto
### projeto baseado no video : https://www.youtube.com/watch?v=XqxUHVVO7-U
## Clonar o repositorio

```git clone git@github.com:j0a03/jaomovie.git``` 

## Entrar na pasta

```cd jaomovie``` 

## Instalar Ferramentas

```npm install``` 

## Iniciar projeto

```npm start```
